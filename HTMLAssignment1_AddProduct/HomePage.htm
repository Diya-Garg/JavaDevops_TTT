HTML & CSS

HTML stands for Hypertext markup language . It is a mark up language which is used to create static web pages . It is a language through which we will specify what we want to display to the user.

CSS stands for Cascading style sheet . It is used to provide styling to the web pages created using CSS. CSS can be categorized into three types:-

  • Inline CSS
  • Internal CSS
  • External CSS
